remote origin add http://...） $ git pull //拉代码 $ git status //查看有什么内容未提交 $ git rm [文件名] 重设回之前的版本： $ git reset --hard HEAD^ // 上一个版本就是HEAD^，上上一个版本就是HEAD^^ $ git reset --hard 1094a // 去任意一个git上存在得版本，输入版本号。版本号没必要写全，前几位就可以了（hard是指回滚掉三个区到指定版本） $ git reset --soft 1094a // soft指回滚归档区，未commit // --mixed指滚回“未add.的状态，显示已 增、改、删 内容的痕迹。” // --soft指回滚到“已增-改-删 ，但未add .环节）* // --hard 指“工作区-缓冲区add-归档区commit”（回退某次所有“增-删-改”文件或内容） $ git revert xxx //commit后，回退前第3次以上所修改的内容，待确认修改新内容（可以在这步时回退上一次的修改内容） 分支： $ git branch //查看branch，就只剩下master分支了 //（git branch -v // 查看分支 / git branch -vv // 查看分支与远程分支的关系） $ git checkout [master] //切换回master分支 创建并切换到新分支： git checkout -b xxx （xxx是分支名） git switch -c / git switch --create 删除分支： git branch -d xxx // 删除 git branch -D xxx // 强行删除 git branch --move old_branch_name new_branch_name // 修改分支名 差异比较： git diff branchXXX // 与某分支进行比较 git diff --staged // 与暂存区的文件进行比较 合并分支操作： $ git merge xxx 重设回之前的版本： $ git reset --hard HEAD^ //
